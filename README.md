# LeaveManagementSystem

Bu projede kullanılan teknolojiler

- Java 1.8
- Spring Boot
- Hibernate
- PostgreSQL:
proje içindeki compose dosyası ayağa kaldırılmalı: docker-compose -f docker-compose.yml up -d
- Swagger
- IDE: Intellij IDEA
